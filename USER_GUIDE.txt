                              QUICK START
================================================================================

1. INSTALLATION & SETUP
   - Install Python 3.8+ on your system
   - Run: pip install -r requirements.txt
   - Copy .env.example to .env and configure your API keys
   - Run: python quick_start.py

2. FIRST TIME SETUP
   - Open http://localhost:8080 in your browser
   - You'll see a clean welcome screen
   - Click "Create New Session" to start your first chat

1. Click the "+" button next to "Chats" in the sidebar
2. Enter a descriptive name (e.g., "Physics Study", "Math Homework")
3. Click "Create Session" or press Enter
4. Your new chat session will open automatically

================================================================================

COMMON ISSUES & SOLUTIONS

1. "AI engine not ready" error
   ‚Üí Check your API keys in the .env file
   ‚Üí Restart the application
   ‚Üí Ensure internet connection for AI services

2. Voice input not working
   ‚Üí Check microphone permissions in browser
   ‚Üí Ensure you're using HTTPS or localhost
   ‚Üí Try refreshing the page

3. Document upload fails
   ‚Üí Check file size (max 50MB)
   ‚Üí Ensure file format is supported
   ‚Üí Try uploading a smaller file first

4. Session not saving messages
   ‚Üí Create a proper session first (don't use default)
   ‚Üí Check browser local storage permissions
   ‚Üí Try refreshing and creating a new session

5. Interface elements not showing
   ‚Üí Clear browser cache and cookies
   ‚Üí Disable browser extensions temporarily
   ‚Üí Try in incognito/private browsing mode
